# ExamenUnidad3_Patrones_HerreraSandovalJoseMiguel
Nombre: Herrera Sandoval Jose Miguel

Nombre Proyecto: Plataforma de cursos con certificados personalizados

Breve descripción del funcionamiento: El presente proyecto muestra una ejecución por consola, que simula una plataforma de certificados personalizados, donde interactúa de forma independiente el Administrador y el Usuario. Se utilizaron Decorator y Bridge como patrones de diseño principales para la estructura. Creado y Ejecutado en Visual Studio Community 2022 (.NET FrameWork 4.7.2).
